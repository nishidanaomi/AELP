# Preparando o Ambiente:

Para rodar um script com sucesso, siga este checklist:

1. **Localização:** O terminal deve estar "dentro" da pasta do arquivo.
   - *Dica:* No VS Code, clique com o botão direito na pasta e escolha **"Open in Integrated Terminal"**.
2. **Caminho (Path):** Verifique se o caminho no terminal termina no nome da sua pasta atual.
3. **Comando:** Digite `node nome-do-arquivo.js` para rodar em qualquer terminal (PowerShell ou Bash).

> **Nota:** Se você tentar rodar o comando fora da pasta, o Node retornará o erro: 
> `Error: Cannot find module`.


---

## Comandos no terminal powershell:

- `dir` (Directory): Lista o conteúdo de um diretório;
- `cd` (Change Directory): Muda de diretório;
- `cd ..`: Volta um nível no diretório;
- `cls` (Clear Screen): Limpa a tela do terminal;
- `node`: Executa um arquivo JavaScript (.js) com o Node.js;

### Atalhos com teclado:

| Atalho = Função |
- `Ctrl + J` = Abre o terminal dentro do VS Code;
- `Ctrl + L` = Comando para limpar a tela do terminal ("cls");
- `Ctrl + C` = Interromper Node.js (caso não funcione digitar "cls" no terminal);
- `↑` (seta para cima) = recupera o último comando digitado;
- `Tab` = Autocompletar nomes de arquivos/pastas;
- `F7` = Exibe histórico de comandos em lista; > não funciona no Bash

---

## No terminal Git Bash

- `pwd` (Print Working Directory): Mostra em qual pasta você está agora (o seu endereço atual);
- `ls` (List): Lista todos os arquivos da pasta (é o dir do Windows);
- `cd` nome-da-pasta (Change Directory): Entra em uma pasta;
- `cd ..`: Volta uma pasta para trás;
- `touch nome-do-arquivo.md`: Cria um arquivo novo;
- `mkdir nome-da-pasta`: Cria uma pasta nova rápida;
- `clear`: Limpa a tela (é o cls do windows);

### Atalhos com teclado:

- `Ctrl + J` = Abre o terminal dentro do VS Code;
- `Ctrl + L` = Comando para limpar a tela do terminal ("cls");
- `Ctrl + C` = Interromper Node.js (caso não funcione digitar "cls" no terminal);
- `↑` (seta para cima) = recupera o último comando digitado;
- `Tab` = Autocompletar nomes de arquivos/pastas;
- `Ctrl + A` = Move o cursor para o início da linha; >não funciona no Shell
- `Ctrl + E` = Move o cursor para o final da linha; >não funciona no Shell
- `Alt + Backspace` = Apaga uma palavra inteira; >não funciona no Shell
